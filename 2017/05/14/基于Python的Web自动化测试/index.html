定位页面元素

一种方式是通过特定元素的API去查询。(Public Methods)

  • find_element_by_id # id号是唯一的，返回的元素要么为空，要么只有一个。
  • find_element_by_name # 存在多个。
  • find_element_by_xpath # 存在多个。
  • find_element_by_link_text # 存在多个。
  • find_element_by_partial_link_text # 存在多个。
  • find_element_by_tag_name # 存在多个。
  • find_element_by_class_name # 存在多个。
  • find_element_by_css_selector # 存在多个。

当未查询到相应元素时，会抛出异常 NoSuchElementException

一种方式是通过By去查询。(Private Methods)
