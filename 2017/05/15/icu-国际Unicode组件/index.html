icu-国际Unicode组件

发表于 2017-05-15 | 分类于 组件 | 0 comments | 阅读次数:
字数统计: 424 字 | 阅读时长 ≈ 1 分钟

ICU是成熟的，广泛使用的一套C / C ++和Java库，为软件应用程序提供Unicode和全球化支持。ICU是广泛可移植的，并在所有平台上以及C / C ++和Java软件之间提供相同的结果。

提供的几个重要服务：

  • 代码页转换 将文本数据转换为Unicode或几何其他字符集或编码。ICU的转换表基于IBM在数十年来收集的字符集数据，并且是任何地方最完整的。
  • Collation 根据特定语言，地区或国家的惯例和标准来比较字符串。
  • 格式 根据所选区域的惯例格式化数字，日期，时间和货币金额。这包括将月份和日期名称翻译成所选择的语言，选择适当的缩写，正确排序字段等。此数据也来自公共语言环境数据存储库。
  • 时间计算 传统公历之前提供了多种类型的日历。提供了一整套时区计算API。
  • Unicode支持 ICU密切跟踪Unicode标准，可轻松访问所有的许多Unicode字符属性，Unicode规范化，大小写转换和其他基本操作由指定的 Unicode标准。
  • 正则表达式 ICU的正则表达式完全支持Unicode，同时提供非常有竞争力的性能。
  • Bidi 支持处理包含从左到右（英文）和从右到左（阿拉伯语或希伯来语）数据混合的文本。
  • 文本边界 找到文本范围内的单词，句子，段落的位置，或者在显示文本时识别适合换行的位置。
