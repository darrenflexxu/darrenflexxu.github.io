如何利用 hexo+github 构建属于你自己的博客

发表于 2016-09-16 | 分类于 hexo | 0 comments | 阅读次数:
字数统计: 288 字 | 阅读时长 ≈ 1 分钟

这篇文章将告诉你如何用 hexo 以及 github的Pages搭建属于你的博客。

准备工作

本地 git shell 以SSH方式连接 GitHub 服务器

ssh -T git@github.com

在 github 上新建一个项目

  1. 每个 github 用户名下面只能建立一个
  2. 项目名称形如：xxx.github.io (xxx为用户名)

本地初始化 Hexo

在 git shell 中输入如下命令：

$ cd /
$ npm install hexo-cli -g
$ cd /
$ hexo init Hexo
$ cd Hexo
$ npm install

利用Hexo创建博客

本地执行 Hexo

$ hexo generate (可以简写成: hexo g)
$ hexo server

部署 Hexo 到 GitHub 上

  1. 配置 Hexo 目录下的 _config.yml 文件，如图所示

  1. 生产静态页面，并同步到 github 服务器中
$ hexo g // 生产静态页面。
$ hexo d // 部署到服务器中。
